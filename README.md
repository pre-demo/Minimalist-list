# 极简待办

一个给自己用的极简待办清单网页，只有添加和勾选功能。

## 功能特点

- ✅ 添加任务：输入框 + 添加按钮，支持回车快捷添加
- ✅ 查看任务：列表显示所有待办，未完成在上，已完成在下
- ✅ 完成任务：点击勾选，显示删除线并移至底部
- ✅ 取消完成：点击已勾选的复选框，移除删除线并移至顶部
- ✅ 数据持久化：使用浏览器 localStorage 存储，刷新页面数据不丢失
- ✅ 空状态提示：无任务时显示友好提示
- ✅ 响应式设计：适配桌面和移动设备

## 技术栈

- HTML5
- CSS3 (Tailwind CSS v3)
- JavaScript (原生)
- Font Awesome 图标

## 使用方法

1. 下载项目文件
2. 直接打开 `index.html` 文件即可使用
3. 无需安装任何依赖，纯前端实现

## 开发说明

### 核心功能实现

1. **数据存储结构**：
  
  ```javascript
  [
  {
   "id": "唯一标识（时间戳）",
   "content": "任务内容",
   "isCompleted": false/true
  }
  ]
  ```
  
2. **本地存储**：
  

- localStorage 键名：`minimal-todo-tasks`
- 自动保存：每次任务状态改变时自动保存

3. **动画效果**：

- 任务添加时的滑入动画
- 任务完成状态切换的平滑过渡

### 主要交互逻辑

1. 添加任务：
  
  - 输入内容 → 点击添加/按回车 → 任务出现在列表顶部
  - 输入框为空时添加按钮禁用
  - 添加成功后输入框自动清空并保持聚焦
2. 完成任务：
  
  - 点击复选框 → 显示删除线 → 移至已完成区域
  - 再次点击 → 移除删除线 → 移回待办区域
3. 数据持久化：
  
  - 所有操作实时保存到 localStorage
  - 页面加载时自动读取并渲染任务列表
